FAQ

In this section you may find Frequently Asked Questions and their answers.
If you still have questions, do not hesitate to contact us. We will be glad to assist you!

  1. What does myWebRTC do?



    2. myWebRTC is a free software based on the WebRTC of the W3C standard. At this stage it does not contain a STUN-server of its own, but uses public STUN servers and it consists of three basic elements:

    • a website:
      for the communication management with a window design for the communication streams,
    • a node.js server:
      for the connection management and
    • a MySQL database:
      for the user management interface.

  2. Which versions of Chrome, Firefox, Opera and Android support the myWebRTC application?



    3. Here you may have an update overview of the Browser versions which support the WebRTC standard.


    Source: caniuse.com

  3. Does myWebRTC work also on iOS mobile platform?



    4. Unfortunately, the present iOS-App for Chrome on iPad and iPhone does not yet support voice and video.

  4. What kind of computer operation systems support the application myWebRTC?



    5. All computer operation systems (Microsoft, Linux, Apple) are compatible and support the application myWebRTC.
    Till now we have tested successfully Windows 7, 8, 8.1 as also Mac OS X Mavericks 9.1, OS X Yosemite 10.10.2.

  5. Is myWebRTC secure?



    6. myWebRTC is a securer replacement for Skype, FaceTime or Google Hangouts, because:

    • All communication data are encrypted.
    • All connections are managed on your own server.
    • It is based on standards.
    • It is free open source.
    • No terms and conditions have to be signed.

  6. Why should I register myself?



    7. The registration on myWebRTC application is not mandatory. You can use the Demo-Version of the application and experience the WebRTC Peer-to-Peer connection.

    However, by registering yourself you will have access to the contact list. That means, that you will be able to see who is online and call them directly.

  7. Is the use of the application for free?



    8. myWebRTC is a license-free open source software!
    That means that you may use the application for free, but you may also extend and even change the source code.

  8. The File Transfer button is not working, why?



    9. Since the myWebRTC application is a part of the Master course Business Computing, the two semester time was not enough for the full development of the application. It is already in the soon plans of the team to develop the application further, by adding the File Transfer feature.

  9. I have read on the brochure that the application offers desktop-sharing feature but now I do not see it, why?



    10. The developers of the myWebRTC team have already programmed the code, but there is a problem with the implementation of this feature.
    • Chrome Browser:
      there is a problem with the new Versions, that the user is not able to activate/deactivate the flags which enable the screen-sharing feature. Therefore it is the installation of an Extension called „Screen Capturing“ required for a non SSL (https) connection.
    • Mozilla Browser:
      as an open source Browser offers the possibility to the users to change the Screen-Sharing settings, but then it is again the installation of an Extension called „Enable Screen Capture“ required.
    • Opera Browser:
      there are no information announced.

    Since the first principle of myWebRTC is "No software or browser plug-in!", the myWebRTC team is trying to find a solution without the installation of any extension.
